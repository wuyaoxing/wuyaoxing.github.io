git 配置多账户

2018-06-22

在工作中，公司会有专门的git账号（公司使用gitlab），有时会做些个人项目放到github，这时候就需要配置多个git账号和服务器。

第一个账号：

  • 设置git账号

  • 创建个人SSH key

  • 把~/.ssh/id_rsa.pub的内容添加到gitlab的SSH keys中

  • 测试

第二个账号：

  • 创建个人SSH key

  • 添加新密钥

  • 修改config文件

  • 把~/.ssh/id_rsa_wyx.pub的内容添加到github的SSH keys中

  • 测试

至此，已完成多个git账号配置，快去clone 项目吧。
